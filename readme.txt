Post Type Slider plugin helps to convert WordPress posts, page, or any custom post type to the slider.

== Description ==
Post Type Slider plugin helps you to create sliders from the different post types. 
With Post Type Slider plugin we can make slider with any post type. No need to worry about jQuery, CSS or images. 
Also, basic structure of the slider will be displayed inside the container in which you want to display the slider. 
so you can easily style the slider based on the design of your theme.  Bx slider is used for the slider, So it will be responsive to all devices. 

Once installed the plugin, a custom post type with the name "Post Type Slider " will be displayed on the admin dashboard. 
From here you can choose which post type you want to convert it into a slider. 
You need to choose the post type, after this, you can able to choose what all are the contents(images, title, contents, Readmore link, and its text) you want to display on the slider. 

There is an option to select whether you want to display the content/excerpt here, also the length of the text also.

Then you can choose bx slider options, how many slides at a time, speed of slides, autoplay, navigation arrows, etc..
Once you publish the slider you can see the shortcode of that slider on the right side of the page.

Shortcode format is like this : [pts_slideshow postid='434']  where post id is the id of slider you are creating.

Just Copy and paste the shortcode wherever you want to place the slider. That's It!.

== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "Post Type Slider".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the Admin Dashboard >> Post Type Slider 

= Admin Installer via zip =

1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select a zip file from your computer.
3. Click the "Install Now" button.
4. Once done uploading, activate Simple Featured Video.

= Manual =
1. Upload the Post Type Slider folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the Admin Dashboard >> Post Type Slider 

After installation and activation of the plugin go to Admin Dashboard >> Post Type Slider

Select the post types , that you want to convert into the slider.

Click on publish and use the displayed shortcode.

== Frequently Asked Questions ==

1. Is it applicable to all post type
	
Yes, you can see all the post type on the drop-down and select the post type you want		

2. What kind of text will be displayed on the slider

You can select all those details while creating the slider from dashbaord. usually post title, content/excerpt, feature image and a read more text will be there.

3. How to generate shortcodes?

Shortcode will be automatically generated based on the settings you have chosen.
